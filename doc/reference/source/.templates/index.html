What authors say:

“Enjoy trying a tool that optimizes an algorithm in a snap!”

OPAL stands for Optimization of Algorithm. OPAL provides a framework in which algorithmic parameter optimization problems can be defined and solved in a programmatic way.

OPAL is written in Python and uses Python as the language in which the parameter optimization problem is described. The procedure consists in three main steps: declare the target algorithm, define an optimization problem and choose a suitable solver.
